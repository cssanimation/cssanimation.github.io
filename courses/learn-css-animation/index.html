About this course

Animation using CSS can be overwhelming to learn all at once. Too much to understand in one go.

With this email course, you’ll learn how, and why to use animation on your web pages. Whether you’re a web designer looking for ways to better communicate your intent, or a web developer wanting to improve your CSS.

Over 20 emails and 4 weeks, each email will cover one particular topic or aspect and set some homework to try. Each email should only take fifteen minutes, and you can proceed at your own pace.

By the end of the month, you will have a solid understanding of CSS animation and be ready to apply animation to your web designs.
