SVG loading spinners

August 17, 2016

[INTRO] - why we have spinners, how they help. Sometimes you want to let people know that you're busy. If you want a generic spinner, services like Spin.js are handy. But they're generic. Let's go further and make something we can tailor to our own brand.

Let's create a spinner we can use on our web sites. The goal will be a single JavaScript file that contains our loading image, animation CSS and logic to switch it on and off as needed.
